# Java-Project
Java-based traffic simulation that uses real streets around the Frankfurt UAS building and communicates in real time with the SUMO simulator. The application visualizes vehicle movements, controls traffic lights and provides insight into traffic flow and behavior

#Requirements:
SUMO: https://sourceforge.net/projects/sumo/ 

#How To Run:
Download Demo1.sumoconfig and run it via sumo-gui
